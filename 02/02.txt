1. AVD와 키패드의 명칭을 정리하시오.
 -  AVD 명칭 : 전원, 볼륨, 화면 회전, 돌아가기, 홈, 오버뷰, 스냅숏, 확장
 - 키패드 명칭 : 돌아가기 버튼, 홈 버튼, 오버뷰 버튼, 상태 바, 타이틀 바
 
2. 다음 각 항목의 단축키는 무엇인가?
 1) AVD 화면을 가로 및 세로로 변경
 2) XML 또는 Java 코드의 자동완성
 3) 자동 임포트
 4) 선택된 영역의 주석(Remark)
 5) 소스코드의 자동 실행
 
3. 안드로이드 프로그래밍을 하다 보면 코딩의 '오류'와 '경고'가 발생한다. 이 두 가지에 대해 설명하시오.
 오류는 스펠링 등 문법적으로 잘못되어 컴파일이 안되는 것이다. 컴파일을 하기 위해서는 이러한 오류를 다 고쳐야 한다. 반면, 경고는 지금 당장 컴파일은 되지만 추후 문제가 될 수 있다는 것을 알려주는 것이다.
 
4. 프로젝트에서 사용하는 리소스가 저장되는 /res 폴더 아래의 하위 폴더와 파일의 용도를 설명하시오.
 - drawble 폴더 : 이미지 파일 저장
 - mipmap : 디자인 화면이나 앱이 설치된 후에 보이는 런처 아이콘
 - xxxhdpi, xxhdpi, xhdpi : 초고해상도 런처 아이콘 파일
 - hdpi 고해상도 런처 아이콘 파일
 - mdpi : 중해상도 런처 아이콘 파일
 - layout : 액티비티를 구성하는 xml 파일 저장, 기본적으로 main.xml이 초기화면으로 지정
 - values : 문자열을 저장하는 strings.xml, 색상표를 저장하는 colors.xml, 스타일을 저장하는 themes.xml 이 있음
 - menu : 메뉴 xml 파일을 저장하며, 필요시 생성해서 사용
 - 그 외 파일 : anim은 애니메이션 저장, 기타 xml 파일을 저장할 xml폴더를 생성할 수 있음
 
5. 완성된 앱이 아이스크림 샌드위치(4.0.x)이상의 스마트폰에서 모두 작동하게 하려면 무엇을 고려해야 하며, 프로젝트를 어떻게 생성해야 하는가?
 Minimum SDK를 고려하여 버전으로 낮춰 프로젝트를 생성한다.
